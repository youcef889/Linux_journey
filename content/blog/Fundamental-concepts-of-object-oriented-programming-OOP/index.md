## Abstraction

## Encapsulation 
Data and the programs that manipulate those data are bound together and their complexity is hidden.
##  Inheritance 
A class can derive methods and properties from another class , resulting in a hierarchy of class .
## Polymorphism 
Different subclass of the same superclass ,can implement their shared interface in different way .
