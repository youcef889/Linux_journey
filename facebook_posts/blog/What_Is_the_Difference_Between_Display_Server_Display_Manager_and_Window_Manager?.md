🖥️ **Display Server vs Display Manager vs Window Manager - Explained!** 🐧🪟

Ever wondered how Linux actually draws your desktop? Here's a clear breakdown of the three core components that make your GUI work.

✅ Display Server (X11/Wayland) — the engine that talks to your hardware
✅ Display Manager (GDM/LightDM/SDDM) — the login screen greeter
✅ Window Manager (i3/KWin/Openbox) — arranges and controls windows
✅ Each plays a distinct role in the Linux graphics stack
✅ Pick the right combo for performance, workflow, or aesthetics
