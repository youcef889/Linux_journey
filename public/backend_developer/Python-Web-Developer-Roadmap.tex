\section*{ЁЯРН Python Web Developer Complete Checklist}

\subsection*{1я╕ПтГг Python Fundamentals}
Learn the basics of Python.

\begin{itemize}[label=$\square$]
    \item Install Python and set up environment
    \item Variables and data types
    \item Numbers and strings
    \item Lists, tuples, sets, dictionaries
    \item Conditional statements (if, elif, else)
    \item Loops (for, while)
    \item Functions
    \item Lambda functions
    \item Modules and packages
    \item File handling
    \item Error handling (try, except)
    \item Object-Oriented Programming (OOP)
    \item Virtual environments (venv)
\end{itemize}

\subsection*{ЁЯМР Web Fundamentals}
Understand how the web works.

\begin{itemize}[label=$\square$]
    \item Basics of HTML
    \item Basics of CSS
    \item Basics of JavaScript
    \item How browsers work
    \item HTTP / HTTPS protocol
    \item Request vs Response
    \item GET / POST / PUT / DELETE methods
    \item Status codes (200, 404, 500)
\end{itemize}

\subsection*{ЁЯФз Version Control}
Learn version control using Git.

\begin{itemize}[label=$\square$]
    \item Install Git
    \item \texttt{git init}
    \item \texttt{git add}
    \item \texttt{git commit}
    \item \texttt{git push}
    \item \texttt{git pull}
    \item Branches
    \item Merge conflicts
    \item Use GitHub
\end{itemize}

\subsection*{ЁЯЧДя╕П Databases}
Learn SQL databases.

\begin{itemize}[label=$\square$]
    \item Install PostgreSQL or MySQL
    \item Database basics
    \item Tables and columns
    \item Primary and foreign keys
    \item SQL queries (SELECT, INSERT, UPDATE, DELETE)
    \item JOIN queries
    \item Indexes
    \item Database relationships
    \item Use SQLite for small apps
\end{itemize}
